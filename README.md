# simple-jigsaw

### To work locally:
1. Clone the repo:
```
git clone https://github.com/marta-fret/simple-fin-app.git
```

2. Install 'yarn' if you don't have it yet
```
npm install --global yarn
```

3. Bootstrap project:
```
yarn bootstrap
```

4. Start development server:
```
yarn start
```

5. To build the project run:
```
yarn build
```

6. To serve the built version:
```
yarn serve
```
